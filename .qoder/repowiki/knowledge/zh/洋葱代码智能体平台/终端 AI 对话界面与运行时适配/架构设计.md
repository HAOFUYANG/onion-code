- 入口层：`App.tsx` 作为根组件，通过 `AssistantRuntimeProvider` 注入本地运行时，并利用 `ThemeProvider` 应用自定义主题。
- 视图层：`components/Thread.tsx` 承载核心对话逻辑，包含首页（HomePage）、消息列表及输入框（Composer），并处理 Slash 命令交互；`components/Dialog.tsx` 提供通用的模态对话框 UI。
- 业务屏层：`screens/ConfigPanel.tsx` 实现多步骤配置向导，用于管理 Python 环境等应用设置。
- 适配层：`runtime/adapter.ts` 实现 `ChatModelAdapter` 接口，将 LangGraph 的流式响应桥接至 assistant-ui 运行时。
- 基础层：`theme/index.ts` 提供终端背景自动检测（dark/light）及语义化色板 `T`，供全模块 UI 组件消费。

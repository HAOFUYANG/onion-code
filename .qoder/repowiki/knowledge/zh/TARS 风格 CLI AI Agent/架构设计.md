- **入口层**：`src/agent/cli.ts` 基于 `commander` 构建交互式终端，支持流式输出与 ESC 中断。
- **核心逻辑**：`src/agent/agent.ts` 利用 `langchain` 和 `@langchain/langgraph` 编排 Agent，通过 `SqliteSaver` 实现多线程上下文记忆。
- **工具系统**：`src/agent/tools/` 封装文件读写、代码执行（JS/Python）、网络搜索等能力，并内置多层安全拦截机制（如禁止危险 Shell 命令）。
- **技能扩展**：`src/agent/skills/` 采用基于目录的发现机制，通过解析 `SKILL.md` 的 YAML frontmatter 动态注入 System Prompt。
- **配置管理**：`src/agent/config.ts` 提供本地 JSON 持久化配置，支持 Python 虚拟环境及 pip 镜像源的自定义。

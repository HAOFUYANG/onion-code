- **UI 与逻辑解耦**：通过 `cli.ts` 作为入口，将命令行解析（Commander）与 UI 渲染（Ink/App.tsx）分离，Agent 核心逻辑（`agent.ts`）独立于界面存在。
- **适配器模式集成**：利用 `@assistant-ui/react-ink` 提供的 `AssistantRuntimeProvider`，通过自定义 `langchainAdapter` 将 LangChain 的执行流桥接至 Assistant-UI 的状态管理模型。
- **跨运行时环境编排**：在 Node.js 主进程中通过 `python_env.ts` 动态管理 Python 虚拟环境，实现了 JS/TS 业务逻辑与 Python 数据科学/处理技能（如 PDF 处理、Excel 分析）的无缝协作。
- **技能驱动架构**：采用 `skills-lock.json` 机制管理外部 AI 技能，配合 `load_skill` 工具实现运行时的能力动态加载与隔离。
